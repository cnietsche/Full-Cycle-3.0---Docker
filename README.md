# Full-Cycle-3.0---Docker
Desafio Full Cycle 3.0 - Docker

___

## Desafio Go

Copie e cole o comando abaixo no terminal.
```
docker run --rm christiannietsche/fullcycle-rocks
```
___

## Nginx com Node.js

Copie e cole o comando abaixo no terminal.
```
docker-compose up -d
```

Acesse o link:

```
http://localhost:8080/
```
